# STO Logistics Telegram Bot 🚛

This is a Telegram bot designed for auto service (STO) centers to track and manage the usage of spare parts.

## 🚀 Functionality
- **Logging:** Mechanics can input the part name, quantity, and vehicle number.
- **Google Sheets Integration:** All data is instantly saved to a designated Google Sheets spreadsheet.
- **Admin Reports:** The administrator receives an instant notification for every new entry.

## 🛠 Technologies Used
- **Python 3.10+**
- **Aiogram 3.x** (Telegram Bot API framework)
- **Gspread** (Python API for Google Sheets)
- **Google Cloud Platform** (Authentication via Service Account)
